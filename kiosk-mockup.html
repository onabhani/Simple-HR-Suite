Kiosk Redesign — Continuous Scan Flow

Optimized for 50+ employee queues — operator sets mode once, camera never stops

Key UX Decisions

  • Operator selects mode once (e.g. "Clock In") — camera stays open continuously
  • Camera never leaves the screen — result flashes as a ~1.5s overlay on top of the live feed
  • No menu return between scans — next employee just steps up and scans
  • Side panel (iPad) / bottom bar (mobile) shows recent scan log so operator can verify
  • STOP button (red) is the only way to exit scan mode — prevents accidental exits
  • Live counter in header shows how many employees have scanned this session
  • Errors flash red briefly and auto-dismiss — doesn't block the queue
