1. In `Settlement_Service::calculate_gratuity()`, change the formula from UAE 21-day to Saudi Article 84:
   - First 5 years: `daily_rate * 15 * years` (was `daily_rate * 21 * years`) -- "half-month per year"
   - After 5 years: `daily_rate * 30 * remaining_years` (this part is already correct -- "full month per year")

2. Add a new method `calculate_gratuity_with_trigger` to Settlement_Service that wraps calculate_gratuity with a trigger_type multiplier:
   ```php
   public static function calculate_gratuity_with_trigger(float $basic_salary, float $years_of_service, string $trigger_type = 'termination'): float {
       $full_gratuity = self::calculate_gratuity($basic_salary, $years_of_service);
       if ($trigger_type === 'resignation') {
           if ($years_of_service < 2) return 0;
           if ($years_of_service < 5) return $full_gratuity / 3;
           if ($years_of_service < 10) return round($full_gratuity * 2 / 3, 2);
           return $full_gratuity; // 10+ years = full
       }
       // termination and contract_end = full gratuity
       return $full_gratuity;
   }
   ```
   Keep the original `calculate_gratuity` method for backward compatibility (it computes full gratuity before trigger adjustment).

3. Add a static method `get_trigger_types()` returning:
   ```php
   return [
       'resignation'  => __('Resignation', 'sfs-hr'),
       'termination'  => __('Termination', 'sfs-hr'),
       'contract_end' => __('Contract End', 'sfs-hr'),
   ];
   ```

4. In `Migrations.php`, after the existing settlements column additions (find the settlement section near line 198+), add:
